# Zilong Liang

This repository contains the source code for [Zilong Liang](https://zlliang.me)'s personal websites. Zilong is a frontend developer at [Booking.com](https://booking.com), previously at Tencent and ByteDance, with a Master's degree in Mathematics from Fudan University.

The monorepo hosts three distinct websites, a custom content management CLI, and shared packages that keep everything consistent.

## Technology stack

All websites are built with [Astro](https://astro.build/) and deployed on [Vercel](https://vercel.com/). The repository is a [pnpm](https://pnpm.io/) workspace monorepo.

| Layer | Technology |
|-------|-----------|
| Framework | Astro 6 |
| Styling | Tailwind CSS 4 |
| Language | TypeScript 5 |
| Package manager | pnpm |
| Deployment | Vercel |
| CLI framework | CAC |
